# Fraud Statistical Signals

## UPI Fraud Statistical Signal Analysis

### Problem Statement
The rapid growth of UPI (Unified Payments Interface) transactions has increased the risk of fraudulent activities. The objective of this project is to analyze UPI transaction data using statistical methods to identify fraud-related patterns and signals that can help detect suspicious transactions.

---

## Dataset Description
The dataset contains UPI transaction records with various attributes such as transaction amount, sender and receiver details, account age, payment method, transaction status, and fraud labels.

The dataset is used to:
- Analyze transaction behavior.
- Compare fraudulent and non-fraudulent transactions.
- Identify statistical differences between normal and fraudulent transactions.

---

## Statistical Methods
The following statistical techniques were applied:

- Descriptive Statistics
- Distribution Analysis
- Correlation Analysis
- Box Plot Analysis
- Frequency Distribution
- Outlier Detection
- Comparative Analysis between Fraud and Non-Fraud Transactions

---

## Fraud Signal Findings

Some important observations include:

- Fraudulent transactions exhibit different transaction amount distributions.
- Certain payment methods show higher fraud occurrence.
- Receiver account age influences fraud probability.
- Statistical outliers are more common in fraudulent transactions.
- Correlation analysis highlights relationships among important transaction features.

---

## Visual Analysis

The project includes visualizations such as:

These visualizations help understand fraud behavior and identify abnormal transaction patterns.
